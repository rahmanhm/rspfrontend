An RSP Game

The Game has two moods - the AutoPlay and the User Decides - Mood. In the Autoplay Version the computer is selecting the tools for both Computer and Human automatically and two of the Three Draws wins the game. In Manuel Mood, the User Picks up their Tool by Clicking them and with that tool will compete against the Computer. There are un limited number of tries. In the Manual Mood the number of Wins will be recorded for each player until the session is reloaded.

To Start, an Application Token should be Provided First! The Token is obtainable by contacting the page owner!
